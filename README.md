# Bing Auto Search

### 消耗瀏覽器記憶體
- 每次自動取得隨機搜尋關鍵字
- 每日自動進行搜尋任務
- 每次自動在搜尋後持續的滾動頁面
- 每日自動載入外部搜尋詞彙列表，可在 `CONFIG` 修改
- 每日搜尋上限與隨機秒數可在 `CONFIG` 修改

### 外部搜尋詞彙範例 example.json
``` json
{
  "keywords": ["程式教學", "作業系統 環境變數"],
  "keywordFix": ["最新", "資訊"],
  "enWordFix": ["英文", "中文"]
}
```
